# PandaiKids homepage living effects assets

This folder is reserved for premium image-based decorative effects on the homepage.

Only add polished transparent-background image assets here. Do not use emoji,
unicode symbols, sticker placeholders, GIFs, videos, canvas, or low-quality
temporary artwork.

Expected optional files:

- `butterfly-3d-01.webp`
- `butterfly-3d-02.webp`
- `bird-white-01.webp`
- `leaf-3d-01.webp`
- `sparkle-soft-01.webp`
- `light-particle-01.webp`

Current rule:

If a high-quality asset is missing, the corresponding effect must not render.
It is better for the homepage to stay clean than to show a cheap placeholder.
