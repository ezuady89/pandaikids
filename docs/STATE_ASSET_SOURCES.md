# State Symbol Sources

The state and federal-territory symbols are stored locally under
`public/assets/states/` for reliable loading.

- Flags: Wikimedia Commons files named `Flag of <state>.svg`
- State crests: Wikimedia Commons files named `Coat of arms of <state>.svg`
- Kuala Lumpur: `Flag of Kuala Lumpur, Malaysia.svg` and `Coat of Arms of Kuala Lumpur.svg`
- Labuan: `Flag of Labuan.svg` and `Seal of Labuan.svg`
- Putrajaya: `Flag of Putrajaya.svg` and `Coat of arms of Putrajaya.svg`

Source collections:

- <https://commons.wikimedia.org/wiki/Category:Flags_of_states_of_Malaysia>
- <https://commons.wikimedia.org/wiki/Category:SVG_coats_of_arms_of_states_of_Malaysia>

Each insignia remains subject to the reuse and legal-status information on its
corresponding Wikimedia Commons file page. These assets are displayed without
modification except responsive sizing.
