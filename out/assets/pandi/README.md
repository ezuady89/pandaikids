# Pandi Character Kit

This folder is the official PandaiKids mascot pose kit.

Use these files instead of random/generated mascot styles:

- `pandi-wave.png` — homepage greeting and friendly welcome
- `pandi-happy.png` — summary, completion and warm success
- `pandi-teacher.png` — wrong answer / gentle teaching mode
- `pandi-thinking.png` — question, waiting and thinking state
- `pandi-celebrate.png` — correct answer and milestones
- `pandi-reading.png` — reading/literacy contexts
- `pandi-explorer.png` — journey/adventure intro
- `pandi-flag.png` — state identity / flag contexts
- `pandi-blindbox.png` — Blind Box reward sequence

Keep one face identity, one rendering style and one body proportion.
Future final artwork can replace these files without changing the code paths.
