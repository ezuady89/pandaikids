# PandaiKids Homepage Full World

Fail ini ialah homepage functional untuk projek Next.js PandaiKids.

Upload/replace ke repo:

- `app/page.tsx`
- `components/homepage/PandaiKidsHomepage.tsx`
- `components/homepage/PandaiKidsHomepage.module.css`

Keperluan aset sedia ada:

- `/public/assets/backgrounds/pandaikids-world-journey-v1.webp`
- `/public/assets/pandi/pandi-wave.png`
- `/public/assets/states/flags/*.svg`

Fungsi yang tersedia:

- Isi nama anak.
- Pilih negeri.
- Pilih umur.
- Simpan profil di localStorage.
- Button Quiz masuk ke `/world/mathematics`.
- Button Dunia masuk ke `/journey`.
- Button Koleksi buka modal.
- Button Ibu Bapa buka modal.
- Mula semula reset profil.

Nota:

Homepage ini bukan fixed wallpaper. Background, Pandi, card dan buttons ialah layer/function sebenar dalam React.
