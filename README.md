# PandaiKids 1.0

Starter code sebenar untuk aplikasi PandaiKids.

## Cara run local

```bash
npm install
npm run dev
```

Buka `http://localhost:3000`.

## Deploy ke Vercel

1. Upload folder ini ke GitHub.
2. Import project di Vercel.
3. Deploy.
4. Connect domain `pandaikids.com`.

## Modul siap dalam starter ini

- Landing page dengan Pandi sahaja
- Onboarding anak
- Home dashboard
- Sahabat Pandai
- Tahun/Subjek/Topik
- Kuiz dengan Amani sebagai teman belajar
- Reward + Pip
- Peta Pengembaraan
- Profil anak
- Struktur code Next.js
