# PandaiKids v1 Starter

Projek pemula yang bersih untuk GitHub + Vercel.

## Cara guna
1. Extract ZIP.
2. Upload semua fail/folder ke repository GitHub `pandaikids`.
3. Vercel akan detect sebagai Vite React.
4. Build command: `npm run build`
5. Output directory: `dist`

## Local test
```bash
npm install
npm run dev
npm run build
```
