# PandaiKids Build 03 - Cinematic Edition

Versi ini fokus pada visual premium Pandi, background lebih hidup, dan flow app yang lebih rasa seperti game mobile.

## Cara upload ke GitHub
1. Extract zip ini.
2. Upload semua fail ke repo GitHub.
3. Pastikan `index.html`, `style.css`, `app.js`, dan folder `assets` berada di root repo.
4. Buka GitHub Pages semula.

## Flow yang boleh klik
- Home → Mula Belajar
- Quiz → Reward
- Reward → Blind Box
- Blind Box → Koleksi
- Home → Koleksi
- Home → Ibu Bapa
